O que é o Seasheep?

O Seasheep é um grupo composto por uma única pessoa (no caso, o criador deste site), especializado em linguagem Python para criação de softwares relacionados a automação de tarefas web com criação de RPAs que podem ser integrados com bases de dados como planilhas Excel ou bancos de dados SQL, podendo estes dados serem devidamente lidos, tratados e manipulados por meio da linguagem ao serem colocados para integração no sistema.

Por que o Seasheep?

O foco do grupo é o poder de automatizar tarefas no qual levariam horas para serem terminadas se feitas de maneira manual, o que poderia acarretar, querendo ou não, em erros indesejados e no pior dos casos, até crítico!

Imagine, por exemplo, que você tenha que cadastrar vários itens que estão inseridos dentro de uma planilha no sistema da sua empresa, digamos que a quantidade desses itens seja de 100, 200 ou, quem sabe, até 500 itens para cadastrar! Agora, e se isso fosse um processo diário? Talvez não só registrar os itens mas mandar um relatório detalhado por meio de arquivos PDF através de um email para outros setores ou até outras empresas? Isso seria um baita trabalho, não é? Pois é aí que a automação de tarefas entra em ação!

Processos que facilmente levariam horas e horas para serem realizadas, podem ser feitas em segundos através de um script bem montado e planejado para o problema ser solucionado, te dando mais tempo para fazer outras coisas - seja aquela reunião que você estava planejando há semanas ou até mesmo para tomar aquele cappuccino na hora do café. E isso tudo, com apenas o conhecimento e a masterização de uma única linguagem de programação versátil e poderosa.

Quais ferramentas são usadas?

Como dito anteriormente, o grupo é especializado em linguagem Python para a realização das tarefas, confira a lista de ferramentas da linguagem que são usadas nos trabalhos:

  • Selenium
  • Playwright
  • PyAutoGUI
  • Pandas
  • Openpyxl
